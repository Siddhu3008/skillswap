\# SkillSwap



Peer-to-peer learning platform built using MERN stack.



\## Features

\- Skill exchange between users

\- Video learning

\- Tutor dashboard

\- Doubt system

\- Reviews and ratings



\## Tech Stack

\- MongoDB

\- Express

\- React

\- Node.js



\## How to Run

1\. Clone repo

2\. npm install

3\. npm run dev
